Privacy Policy for Lucy Bridge

Lucy Bridge is a Chrome extension that captures the currently active browser tab content when the user clicks the extension icon.

Data Collection

We do not collect, store, or sell personal data. The extension only accesses webpage content temporarily to send it to the local Lucy desktop application for processing.

Data Usage

All data captured from webpages is used locally and is not transmitted to any third-party servers.

Third Parties

No data is sold, shared, or transferred to third parties.

Contact

For questions, open an issue on the GitHub repository.

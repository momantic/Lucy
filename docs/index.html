What is Lucy?

Lucy is not a chatbot. She is a multi-step AI agent system that can: browse, analyze, and generate structured outputs using local MLX models.

Capabilities

🧠 Local MLX reasoning engine
🌐 Browser Bridge web research
✍️ LinkedIn post generation
⚙️ Multi-agent workflow system

How Lucy thinks

Browser → Extract → Analyze → Write → Review → Output

Installation

  1. Download Lucy
  2. Open Lucy.app (right-click → Open)
  3. Install Chrome Browser Bridge extension
  4. Start local bridge server
  5. Ask Lucy anything

Example

write me a linkedin post about ai agents in stock markets

What is Lucy?

Lucy is a local-first AI desktop agent that runs on your Mac. She can browse the web, read LinkedIn pages, analyze content, and write structured posts using local MLX models.

🧠 What she does

  • Reads web pages via Browser Bridge
  • Generates LinkedIn posts
  • Runs local AI models (MLX)
  • Uses multi-step agent workflows
  • Works completely locally

🔒 Privacy

  • No cloud API required
  • No data upload by default
  • Browser data stays local
  • Runs on your machine

⚙️ How to install

  1. Download Lucy
  2. Open Lucy.app (right-click → Open)
  3. Install Chrome extension (Browser Bridge)
  4. Enable developer mode in Chrome
  5. Load unpacked extension from Lucy folder

🌐 Example prompt

write me a linkedin post about ai agents in stock markets

🚀 Architecture

Lucy uses a multi-agent pipeline: Browser Bridge → MLX analysis → Writer agent → Reviewer → Final post
